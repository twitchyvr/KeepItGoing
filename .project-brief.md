# Project Brief — KeepItGoing

- KeepItGoing is a macOS automation tool for Claude Code productivity
- Target platform: macOS (Apple Silicon + Intel), iTerm2
- License: TBD — do NOT choose a license without asking Matt
- This may become open-source after sanitization review — treat all code as potentially public
- No external Python dependencies — stdlib only for generator and state writer
- The AppleScript source must NOT contain hardcoded user paths — install.sh patches them at build time
- Stack detection must be comprehensive — support as many stacks as possible (currently 84)
- Directive categories should be high-quality, varied, and actionable — not generic filler
- Anti-derail preamble is mandatory on every prompt — the AI must not context-switch
- Every prompt ends with "think deeply" + "fill gaps" permanent frame
- The `kig` CLI must remain portable — no dependencies beyond bash and iTerm
